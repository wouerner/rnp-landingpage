INTRODUCTION
------------

Views plugin which redirect views display module depending current theme.

REQUIREMENTS
------------

* Views.

INSTALLATION
------------
 
 * Install as you would normally install a contributed Drupal module. 
   See: https://drupal.org/documentation/install/modules-themes/modules-7
   for further information.

CONFIGURATION
-------------

* Set "Trigger theme" and "Display redirect from" in views "Theme" sections

TROUBLESHOOTING
---------------

* Run "drush cc views" if you have problems while uninstall module.
